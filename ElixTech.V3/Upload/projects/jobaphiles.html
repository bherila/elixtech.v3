Jobaphiles.com

Jobaphiles is a job-auction web site powered by asp.net. In May 2008, I worked with the client to rapidly design and implement a modular prototype with only basic functionality. Throughout summer 2008, we worked together to refine the project and add features.

Jobaphiles continued to grow, and two additional developers joined the team in Fall 2008. Today, I manage Jobaphiles as Chief Technology Officer.

Features:

  • Object-relational mapping system in C# supporting both Microsoft SQL Server and MySql 5.
  • Grid-ready; handles thousands of users every day.
